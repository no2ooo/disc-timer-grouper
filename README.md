# 飞盘计时器 + 随机分组 · 微信小程序

一套为飞盘社群设计的轻量小程序,**纯本地运行、无需登录、无需后端**,打开即用。

## 📌 项目简介

「飞盘计时器 + 随机分组」是一款面向飞盘社群组织者(如大连飞盘社群)的轻量工具小程序。它把训练/比赛现场最常需要的几件事——**节奏把控、分组安排、名单管理**——浓缩进一个 App,且**全程本地运行、无需登录、无需后端服务器**,打开即用、数据留在自己手机上。

- **给谁用**:飞盘社群组织者、队长、教练,以及任何需要快速分组或计时提醒的运动场景
- **解决什么**:训练间歇计时、随机均分队伍、报名名单智能解析与性别均衡分组
- **设计取向**:暗黑高对比配色(荧光黄主题),强光户外也清晰;操作尽量一步到位,减少现场手忙脚乱

## ✨ 功能

### 1. 翻页时钟(首页)
- 仿复古翻页时钟,带真正的 3D 翻页动画(旧数字翻走、新数字翻入)
- 自适应横竖屏:横屏时数字按屏幕比例缩放不溢出,并自动隐藏底部导航栏
- 作为训练/比赛开场前的节奏看板

### 2. 计时器
- 3 种模式,点右上角 `↻` 一键切换:
  - **高强度间歇训练**(参考 App 风格:黄/绿/红大色块)
  - **周期**(单 phase 循环,适合跑步/拉伸)
  - **秒表**(可设置每 N 秒响一次,适合折返跑)
- 视觉:黑底 + 荧光黄 + 浅绿,大数字 200rpx 复刻原图风格
- 点齿轮 → 弹菜单调整参数,可保存为预设
- 振动 + 短促提示音(无需音频资源,base64 嵌入 1KB 内)
- 屏幕常亮可选,运动时不会黑屏

### 3. 随机分组
- 多行输入人名(支持中文逗号、空格、英文逗号分隔)
- 一键平均分 3 组
- **人数不够时,最后一组兜底**(前两组永远均匀,余数并入第 3 组)
- 长按分组卡可复制单组/全部
- 历史记录最近 20 次分组

### 4. 我的
- 查看已保存的训练预设,点击应用 → 自动跳到计时页

### 5. 设置
- 声音 / 振动 / 屏幕常亮 / 闪屏提醒 / 允许旋转
- 一键清除全部数据

## 🚀 如何运行

1. 打开[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目 → 选择本目录(`飞盘计时器+随机分组`)
3. AppID 选**测试号**即可
4. 点击"编译"即可预览

## 📁 目录结构

```
├── app.js / app.json / app.wxss     全局配置
├── pages/

│   ├── storage.js  本地预设/历史持久化
│   └── notify.js   振动 + base64 提示音
├── project.config.json

```

## 🎨 设计要点

- **零第三方依赖**:不引入任何 npm 包,主包轻量、编译快、过审风险低
- **图标与提示音用本地资源**:tabBar 图标为 `assets/` 下 10 张本地 PNG,提示音用本地 `qidi.mp3`,均无需联网
- **零后端**:全部数据存 `wx.storage`,首次启动自动初始化
- **暗黑高对比度**:运动场景下强光下也清晰

## 📋 上线清单(过审提示)

- 类别选 **工具 → 效率** 或 **运动健身**
- 类目需要资质:健身类如做"HIIT 训练计时"等需上传《增值电信业务经营许可证》或 ICP
- 备选方案:以"效率"类目上线,避免健身资质要求
- 服务条款 / 隐私协议需在小程序内放置入口(简单链接到 H5 即可)
- 微信小程序对未登录 + 无后端的工具类非常友好,**通过率较高**
